Funciones [lineales]

Una función es una relación especial entre dos conjuntos, llamados dominio y rango. A cada elemento en el dominio le corresponde uno y solo un elemento en el rango. Imagina una máquina expendedora de refrescos: si introduces una moneda (dominio), obtienes una bebida (rango). Cada moneda produce un único resultado.

Notación

Para representar una función, utilizamos la notación \(f(x)\), donde "f" es el nombre de la función y "x" es la variable de entrada. La función toma "x" como entrada y produce un resultado correspondiente, denotado como \(f(x)\). Por ejemplo, \(f(x) = 2x\) representa una función que duplica el valor de "x".

Gráficos de funciones

Los gráficos son una forma visual de representar funciones. El eje x representa la entrada (dominio) y el eje y representa el resultado (rango). Cada punto en el gráfico corresponde a un par de valores \( (x, f(x)) \).

Propiedades

Las funciones tienen varias propiedades que pueden ser útiles para analizarlas.

  • Función Inyectiva (o Inyectividad):
    Una función es inyectiva si cada elemento del dominio se asigna a un único elemento en el rango, es decir, ningún elemento del dominio se asigna a más de un elemento en el rango. Formalmente, una función \(f: A \rightarrow B\) es inyectiva si para cada par de elementos distintos \(a_1\) y \(a_2\) en \(A\), sus imágenes bajo \(f\) son distintas: \(f(a_1) \neq f(a_2)\).

  • Función Sobreyectiva (o Sobreyectividad):
    Una función es sobreyectiva si cada elemento en el rango tiene al menos un elemento en el dominio que se asigna a él. En otras palabras, no hay "huecos" en la asignación de elementos del dominio al rango. Formalmente, una función \(f: A \rightarrow B\) es sobreyectiva si para cada elemento \(b\) en \(B\), hay al menos un elemento \(a\) en \(A\) tal que \(f(a) = b\).

  • Función Biyectiva (o Biyectividad):
    Una función es biyectiva si es tanto inyectiva como sobreyectiva. Esto significa que cada elemento en el dominio se asigna a un único elemento en el rango y que cada elemento en el rango está relacionado con un único elemento en el dominio. En resumen, una función es biyectiva si es una correspondencia uno a uno entre los conjuntos. Formalmente, una función \(f: A \rightarrow B\) es biyectiva si para cada elemento \(b\) en \(B\), hay exactamente un elemento \(a\) en \(A\) tal que \(f(a) = b\), y para cada elemento \(a\) en \(A\), hay exactamente un elemento \(b\) en \(B\) tal que \(f(a) = b\).

Evaluación de funciones

Para evaluar una función en un valor particular de "x", simplemente sustituye ese valor en la expresión de la función. Por ejemplo, si tienes \(f(x) = 3x + 5\) y quieres encontrar \(f(2)\), sustituye "x" por 2: \(f(2) = 3 \times 2 + 5 = 11\).

Ejemplo de evaluación de funciones

Supongamos que tenemos la función \(f(x) = 2x + 3\). Vamos a evaluar \(f\) en varios valores de \(x\) y representar el resultado como pares ordenados: \[ f = \{(x, f(x)) \mid x \in \mathbb{R}\} \]

  • Para \(x = 1\): \(f(1) = 2 \times 1 + 3 = 5\), por lo tanto, \((1, 5)\) está en \(f\).
  • Para \(x = 0\): \(f(0) = 2 \times 0 + 3 = 3\), por lo tanto, \((0, 3)\) está en \(f\).
  • Para \(x = -1\): \(f(-1) = 2 \times (-1) + 3 = 1\), por lo tanto, \((-1, 1)\) está en \(f\).
Estos pares ordenados representan la relación entre \(x\) y \(f(x)\) para la función \(f(x) = 2x + 3\).

Tipos de funciones

Funciones constantes

Una función constante tiene la forma \(f(x) = c\), donde "c" es un número constante. El gráfico de una función constante es una línea horizontal, ya que el resultado es el mismo para cualquier valor de "x".

Ejemplo: Función constante

Ahora consideremos la función \(g(x) = 4\). Para cualquier valor de "x", \(g(x)\) siempre será 4. Esto se muestra en un gráfico como una línea horizontal en \(y = 4\).

Funciones lineales

Una función lineal tiene la forma \(f(x) = mx + b\), donde "m" es la pendiente y "b" es la intersección con el eje y (cuando \(x = 0\)). La pendiente representa la inclinación de la línea en el gráfico.

Ejemplo: Función lineal

Supongamos que tenemos la función \(f(x) = 2x + 3\). Podemos encontrar \(f(1)\) sustituyendo \(x = 1\): \(f(1) = 2 \times 1 + 3 = 5\). Este proceso se puede repetir para cualquier valor de \(x\).
